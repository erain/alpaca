# Alpaca

![Alpaca](https://s3-ap-southeast-1.amazonaws.com/files.yiyu.me/alpaca.jpg)

Alpaca is an project for exploring and testing different components of distibuted systems, including performanace test or network partition tolerating. Right now the components on the are:
- [ZooKeeper](http://zookeeper.apache.org/)
- [RabbitMQ](https://www.rabbitmq.com/)
- [Cassandra](http://cassandra.apache.org/)
- [HBase](https://hbase.apache.org/)
- [twenproxy](https://github.com/twitter/twemproxy)
- [Kafka](http://kafka.apache.org/)
